Portfólio single-page — tema preto & azul, navbar lateral fixa e interativa (estilo Fragoso)
Como usar:
1. Descompacte o ZIP.
2. Abra index.html no navegador ou rode um servidor (python -m http.server 8000).
3. Substitua imagens em assets/fotos/ por suas imagens reais.
4. Atualize links de repositório nos cartões de projeto.
